# Assets Directory

This directory contains all the assets for the FinNest app.

## Required Assets

Please add the following assets to this directory:

1. **icon.png** - App icon (1024x1024 px)
2. **splash.png** - Splash screen image (1284x2778 px for iOS)
3. **adaptive-icon.png** - Android adaptive icon (1024x1024 px)
4. **favicon.png** - Web favicon (48x48 px)

## Design Guidelines

- Use the FinNest color scheme:
  - Deep Navy Blue: #0A2540
  - Gold: #FFD700
- Maintain high resolution for crisp display
- Follow platform-specific guidelines for icons
- Consider using the "FN" logo with gradient background

## Logo Concepts

The logo should reflect:
- Financial stability and growth
- Modern, futuristic design
- Trust and professionalism
- The "nest" metaphor (building and protecting wealth)
